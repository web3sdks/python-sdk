# contracts.token

Interface for interacting with a token contract

<a id="contracts.token.Token"></a>

## Token Objects

```python
class Token(ERC20)
```

Create a standard crypto token or cryptocurrency.

```python
from web3sdks import Web3sdksSDK

# You can customize this to a supported network or your own RPC URL
network = "mumbai"

# Now we can create a new instance of the SDK
sdk = Web3sdksSDK(network)

# If you want to send transactions, you can instantiate the SDK with a private key instead:
#   sdk = Web3sdksSDK.from_private_key(PRIVATE_KEY, network)

contract = sdk.get_token("{{contract_address}}")
```

Mint tokens to a list of wallets.

```python
    TokenAmount("{{wallet_address}}", 1),
    TokenAmount("{{wallet_address}}", 2),

```

**Arguments**:

- `args`: list of wallet addresses and amounts to mint

**Returns**:
